# CodePath TIP101: Intro to Technical Interview Prep

## Overview

This repository contains my coursework, notes, coding challenges, practice problems, and solutions completed as part of **CodePath's TIP101: Intro to Technical Interview Prep** course.

TIP101 is an introductory technical interview preparation program focused on developing strong problem-solving skills through Python programming, data structures, algorithms, and interview communication techniques.

The course follows CodePath's adaptive learning model, allowing students to progress through units at their own pace while building the foundational skills required for technical interviews.

## UMPIRE Framework

Throughout the course, coding problems are solved using CodePath's UMPIRE framework:

### U — Understand

Clarify the problem requirements and constraints.

### M — Match

Identify patterns, algorithms, or data structures that fit the problem.

### P — Plan

Develop a solution strategy before coding.

### I — Implement

Write the solution code.

### R — Review

Verify correctness and test edge cases.

### E — Evaluate

Analyze runtime and space complexity.
